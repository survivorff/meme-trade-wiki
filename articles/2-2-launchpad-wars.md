# 发射台之战：Pump.fun vs LetsBonk vs Moonshot vs 其他

> 发射台是 Meme 币生态的源头。谁控制了发射台，谁就控制了整个市场的入口。

## 表面认知

"发射台不就是发币的地方吗，有什么区别？" 区别大了。

## 真实情况

### 竞争格局演变

**2024 年：Pump.fun 一家独大**
- 几乎没有竞争对手
- 所有 Solana 上的 Meme 币都从这里发射

**2025 年上半年：挑战者出现**
- LetsBonk 依托 BONK 社区快速崛起
- 2025 年 6-7 月一度在日交易量和收入上超越 Pump.fun
- Moonshot 走差异化路线，主打移动端和法币入金

**2025 年下半年至今：Pump.fun 反击**
- 推出 PumpSwap（自有 DEX），垂直整合
- 发行 $PUMP Token，ICO 融资 $13 亿
- 通过 Token 回购（65.4 万 SOL，约 $1.3 亿）创造需求
- 重新夺回 >90% 的市场份额

### 为什么 Pump.fun 很难被颠覆

发射台的竞争本质上是**网络效应的竞争**：

```
更多 Token 创建 → 更多交易者来 → 更多交易量
      ↑                                │
      └────── 更多创建者被吸引 ←────────┘
```

后来者要打破这个循环，通常只有两种方式：
1. **价格战**：更低的费用（LetsBonk 的策略）
2. **差异化**：不同的用户群体或使用场景（Moonshot 的移动端策略）

但 Pump.fun 可以随时跟进降费，而且它的生态绑定（所有交易平台优先支持）让切换成本很高。

### 发射台的技术差异

虽然表面上看都是"一键发币"，但底层实现有重要差异：

- **Bonding Curve 参数**：决定了价格上涨的速度和毕业阈值
- **毕业后的流动性去向**：Pump.fun 迁移到自有的 PumpSwap，LetsBonk 迁移到 Raydium
- **反 Rug 机制**：LP Token 是否销毁、创建者权限是否撤销
- **创建者激励**：交易量分成比例不同

## 经验教训

发射台之战告诉我们：
1. **先发优势 + 网络效应 = 极强的护城河**
2. **垂直整合是终局** —— Pump.fun 推出 PumpSwap 就是为了控制全链路
3. **挑战者需要找到不同的战场** —— 正面竞争几乎不可能赢
