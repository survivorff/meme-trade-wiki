# 一个 Meme 币的一生：从诞生到归零的 5 分钟

> 每天有数万个 Meme 币被创建，99% 在几分钟内死亡。这篇文章带你走完一个 Token 从出生到死亡的完整旅程。

## 表面认知

"有人发了个币，涨了，然后跌了。" 大多数人对 Meme 币生命周期的理解就这么简单。

## 真实情况

一个 Meme 币的生命周期远比你想象的复杂。每个阶段都有不同的参与者在博弈，每个阶段的规则都不一样。

### 阶段一：创建（第 0 秒）

在 Pump.fun 上，创建一个 Token 只需要：
- 一个名字和图片
- 不到 $2 的创建费用
- 30 秒的操作时间

创建完成后，Token 立即进入 **Bonding Curve 阶段**。

**Bonding Curve 是什么？**

简单说，就是一个自动定价的数学公式：
- Token 刚创建时价格极低
- 每有人买入，价格自动上升
- 每有人卖出，价格自动下降
- 不需要传统的做市商或流动性提供者

这个阶段的市值通常在 $0 到 $69K 之间（Pump.fun 的毕业阈值）。

### 阶段二：Bonding Curve 交易（第 0 秒 - 第 N 分钟）

Token 创建后，立即可以在 Pump.fun 上交易。这个阶段的特点：

- **流动性极低**：几百美元的买入就能让价格翻倍
- **狙击手活跃**：机器人在 Token 创建的毫秒级时间内自动买入
- **信息战**：创建者在社交媒体上制造热度，吸引更多买家
- **高度投机**：大多数买家的目标是在几分钟内翻倍卖出

大多数 Token 在这个阶段就死了 —— 没有足够的买盘推动价格到毕业阈值。

### 阶段三：毕业（关键转折点）

当 Bonding Curve 上的市值达到约 $69K 时，Token "毕业"：
- Pump.fun 自动将流动性迁移到 DEX（之前是 Raydium，现在是 PumpSwap）
- 流动性池被创建并锁定（LP Token 被销毁）
- Token 从此可以在开放的 DEX 上自由交易

**只有不到 2% 的 Token 能毕业。** 这是第一道生死线。

### 阶段四：DEX 交易（毕业后）

毕业后，Token 进入更广阔的市场：

- 被 DEXScreener、Birdeye 等数据平台收录
- 交易平台（Axiom、GMGN 等）开始展示和支持交易
- 流动性更深，但也意味着需要更大的资金量来推动价格

这个阶段的典型模式：

```
毕业 → 短暂回调（早期买家获利了结）
     → 如果有持续热度 → 第二波拉升 → 可能上 CEX
     → 如果热度消退 → 缓慢阴跌 → 归零
```

### 阶段五：爆发或死亡

极少数 Token（<0.1%）会经历真正的爆发：
- 被知名 KOL 提及
- 登上 CEX（中心化交易所）
- 形成真正的社区文化（如 BONK、WIF、PEPE）

绝大多数 Token 的结局：
- 创建后几分钟内无人问津 → 直接死亡
- 毕业后短暂交易 → 流动性枯竭 → 缓慢归零
- 被 Rug Pull → 瞬间归零

### 时间线可视化

```
创建 ──→ Bonding Curve 交易 ──→ 毕业？──→ DEX 交易 ──→ 爆发或归零
 │            │                   │           │
 │     大多数死在这里          只有~2%      极少数存活
 │     （无买盘）            能到这里      超过一周
 │
 狙击手在这里等着
```

### 每个阶段的参与者

| 阶段 | 主要参与者 | 他们在做什么 |
|------|-----------|-------------|
| 创建 | 发币者 | 创建 Token，准备营销素材 |
| Bonding Curve | 狙击机器人、早期投机者 | 抢先买入，博短期暴涨 |
| 毕业前后 | 交易平台用户、跟单者 | 追热度买入，或跟踪聪明钱包 |
| DEX 交易 | 散户、KOL 粉丝 | 看到社交媒体推荐后买入 |
| 后期 | 做市商、套利者 | 提供流动性或做价差套利 |

## 经验教训

从平台方的视角看，Meme 币的生命周期本质上是一个**注意力衰减曲线**。Token 的价格和注意力高度正相关 —— 一旦没有新的注意力流入，价格必然下跌。

理解这一点，你就理解了为什么：
- 狙击手要抢在第一时间买入（注意力最集中的时刻）
- KOL 喊单如此有效（注入新的注意力）
- 99% 的 Token 归零（注意力是有限资源）
