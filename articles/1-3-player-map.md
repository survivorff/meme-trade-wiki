# Meme 币市场的参与者图谱

> 在这个市场里，每个人都觉得自己是猎人，但大多数人其实是猎物。

## 表面认知

"就是一群人在买卖 Meme 币。" 实际上这个市场有一条完整的食物链。

## 真实情况

Meme 币市场的参与者可以分为**基础设施层**、**中间层**和**用户层**三个层级。

### 食物链全景

```
┌─────────────────────────────────────────────┐
│              基础设施层（收租者）               │
│  Solana 验证者 · Jito · RPC 服务商 · DEX      │
├─────────────────────────────────────────────┤
│              平台层（服务商 + 收费者）           │
│  发射台 · 交易平台 · 数据平台 · 钱包           │
├─────────────────────────────────────────────┤
│              专业玩家层（信息差获利者）          │
│  狙击手 · MEV 机器人 · 做市商 · Cabal          │
├─────────────────────────────────────────────┤
│              用户层（流动性提供者）              │
│  散户 · KOL 粉丝 · 跟单者 · 新手              │
└─────────────────────────────────────────────┘
                    ↑
              钱从下往上流
```

### 各角色详解

#### 基础设施层

**Solana 验证者**
- 处理所有交易，收取基础交易费
- 通过 Jito 收取 MEV 小费
- 无论市场涨跌，只要有交易就有收入

**Jito**
- Solana 上的 MEV 基础设施，市场份额约 95%
- 提供 Bundle 服务（打包交易，保证执行顺序）
- 运营 Block Engine（交易排序拍卖）
- 2025 年 MEV 捕获超过 $7.2 亿

**RPC 服务商**（Helius、QuickNode、Triton 等）
- 提供节点访问服务，是所有链上操作的入口
- 按请求量收费，交易越多收入越高
- 高级服务（Staked Connection、专用节点）价格更高

**DEX**（Raydium、Jupiter、PumpSwap、Meteora）
- 提供链上交易撮合
- 收取交易手续费（通常 0.25%-1%）
- 流动性提供者赚取手续费分成

#### 平台层

**发射台**（Pump.fun、LetsBonk、Moonshot）
- 提供一键发币服务
- 收取创建费 + Bonding Curve 阶段的交易手续费
- Pump.fun 累计收入超过 $10 亿

**交易平台**（Axiom、GMGN、Photon、BullX）
- 提供交易界面、数据分析、自动化工具
- 收取每笔交易 0.5%-1% 的手续费
- 部分平台还从 MEV 中获利
- Axiom 峰值月收入 $4900 万

**数据平台**（DEXScreener、Birdeye）
- 提供行情数据、图表、新币发现
- 通过广告、高级功能订阅变现

**钱包**（Phantom、Backpack）
- 用户资产的入口
- 内置 Swap 功能收取手续费
- Phantom 月收入约 $2200 万

#### 专业玩家层

**狙击手（Sniper Bot）**
- 在新 Token 创建的毫秒级时间内自动买入
- 通过速度优势获得最低价格
- 使用 Jito Bundle 保证交易优先执行
- 盈利模式：低买高卖，依赖速度优势

**MEV 机器人**
- **三明治攻击者（夹子）**：在用户交易前后插入自己的交易获利
- **套利者**：在不同 DEX 之间做价差套利
- 2025 年三明治攻击从 Solana 用户身上提取了约 $5 亿

**做市商**
- 在 DEX 上提供流动性
- 通过买卖价差获利
- 部分做市商与项目方合作，协助控盘

**Cabal（小圈子）**
- 一群人秘密协调，提前买入某个 Token
- 然后通过 KOL 喊单、社交媒体炒作拉高价格
- 在散户追高时出货获利
- 详见第七章

#### 用户层

**散户**
- 市场中数量最多的参与者
- 信息获取最慢，通常在价格已经涨起来后才看到
- 是整个食物链的最终流动性来源

**KOL 粉丝**
- 跟随 KOL（意见领袖）的推荐买入
- 不知道 KOL 可能已经提前建仓
- 本质上是 KOL 的退出流动性

**跟单者**
- 使用平台的跟单功能，自动复制"聪明钱包"的交易
- 存在延迟问题 —— 跟单者的买入价格通常比被跟踪者高
- 被跟踪的钱包可能会利用跟单者作为退出流动性

### 谁在赚谁的钱

```
散户的钱 → 交易手续费 → 平台
散户的钱 → 滑点 + MEV → 夹子/套利者
散户的钱 → 高位接盘 → 早期买家/Cabal/KOL
所有人的钱 → 基础交易费 → 验证者
所有人的钱 → Jito Tip → 验证者 + Jito
所有人的钱 → RPC 费用 → 节点服务商
```

## 经验教训

这个市场的残酷真相是：**大多数散户是整个生态的"燃料"**。他们的资金通过各种渠道流向基础设施提供者、平台方和专业玩家。

这不是说散户完全没有赚钱的可能 —— 确实有人通过信息优势、速度优势或纯粹的运气赚到了钱。但从统计学角度看，这是一个负和游戏（考虑到各层的手续费抽成）。

理解这个食物链，是你在这个市场生存的第一步。
