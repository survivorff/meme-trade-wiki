# 术语表

> Meme 交易世界的专业术语速查。持续更新中。

---

## 基础概念

| 术语 | 英文 | 解释 |
|------|------|------|
| Meme 币 | Meme Coin | 基于网络文化/梗创建的加密货币，通常没有实际功能，价值由社区情绪驱动 |
| 发射台 | Launchpad | 允许任何人一键创建和发行新 Token 的平台，如 Pump.fun |
| Bonding Curve | Bonding Curve | 一种数学定价曲线，Token 价格随购买量自动上升，无需传统做市商 |
| 毕业 | Graduation | Token 在 Bonding Curve 上达到市值阈值后，流动性自动迁移到 DEX 的过程 |
| 流动性池 | Liquidity Pool (LP) | DEX 上由两种 Token 组成的交易对资金池，提供自动化交易 |
| 地板价 | Floor Price | Token 当前的最低交易价格 |

## 交易相关

| 术语 | 英文 | 解释 |
|------|------|------|
| 滑点 | Slippage | 预期成交价与实际成交价之间的差异，流动性越差滑点越大 |
| Gas 费 | Gas Fee | 在区块链上执行交易需要支付的网络费用 |
| Priority Fee | Priority Fee | Solana 上为提高交易优先级而额外支付的费用 |
| Jito Tip | Jito Tip | 通过 Jito 协议支付给验证者的小费，用于获得交易排序优先权 |
| 狙击 | Sniping | 在新 Token 上线的第一时间自动买入，抢在其他人之前 |
| 跟单 | Copy Trading | 自动复制某个钱包地址的交易行为 |
| 止盈/止损 | Take Profit / Stop Loss | 预设价格条件，达到后自动卖出 |
| DCA | Dollar Cost Averaging | 分批买入/卖出策略，降低单次交易的价格风险 |

## MEV 相关

| 术语 | 英文 | 解释 |
|------|------|------|
| MEV | Maximal Extractable Value | 通过控制交易排序可以提取的最大价值 |
| 三明治攻击 | Sandwich Attack | 攻击者在目标交易前后各插入一笔交易，从价格变动中获利 |
| 抢跑 | Frontrunning | 在目标交易之前插入自己的交易以获利 |
| Bundle | Bundle | 通过 Jito 提交的一组打包交易，保证原子性执行 |
| 夹子 | Sandwich Bot | 执行三明治攻击的自动化机器人 |

## 风险相关

| 术语 | 英文 | 解释 |
|------|------|------|
| Rug Pull | Rug Pull | 项目方突然撤走流动性或抛售 Token，导致价格归零 |
| 蜜罐 | Honeypot | 看似可以买入但无法卖出的恶意 Token 合约 |
| Cabal | Cabal | 一群人秘密协调，通过内幕信息和协同操作操纵 Token 价格 |
| 刷量 | Wash Trading | 同一实体在自己的多个账户之间反复交易，制造虚假交易量 |
| Dev 钱包 | Dev Wallet | Token 创建者持有的钱包，如果持仓比例过高是风险信号 |
